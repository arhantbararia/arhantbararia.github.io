  • GoSnoop

    GoSnoop is a powerful, API-driven reconnaissance framework built in Go. It automates the process of gathering intelligence on web assets by orchestrating a suite of popular open-source security tools. The system exposes RESTful endpoints to initiate scans, manage assets, and retrieve collected data, making it an ideal backend for a security operations dashboard.
